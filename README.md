# Retail-Operation-Sales-Analysis

## Overview
End-to-end Retail Store Operations data
covering Jan–Dec 2023 | 5,000 records | 20 stores

## Tools Used
- MS Excel (SUMIF, COUNTIF, VLOOKUP, Pivot Tables)
- Power Query (Data Cleaning — 30 issues fixed)
- Power BI (3-page interactive dashboard)

## Dataset
- 5,000 rows | 20 columns
- Metro + Tier 2 + Tier 3 cities
- 6 categories | Real Indian brands

## What I Did
1. Cleaned messy raw data using Power Query
2. Fixed duplicates, wrong dates, spelling errors, outliers
3. Built MIS Report with KPIs, store performance, category analysis
4. Built 3-page Power BI dashboard with slicers

## Dashboard Pages
- Page 1: Sales Analysis (KPIs, Monthly Trend, Store Performance)
- Page 2: Store Analysis
- Page 3: Product Analysis
